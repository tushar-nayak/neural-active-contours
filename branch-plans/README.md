# Branch Plans

This directory contains branch-specific documentation for the four experiment
paths created from `polyp-snakeenergy`.

## Branches

- [`unet-baseline`](./unet-baseline/README.md): supervised U-Net baseline.
- [`hybrid-contour-regularizer`](./hybrid-contour-regularizer/README.md): U-Net
  plus contour regularization.
- [`pseudo-label-segmentation`](./pseudo-label-segmentation/README.md):
  pseudo-label-driven segmentation.
- [`self-supervised-pretrain`](./self-supervised-pretrain/README.md):
  self-supervised pretraining followed by fine-tuning.

Each branch directory also includes an `approach.txt` with the immediate work
plan for that branch.
